What is NeoBrutalism?

Neubrutalism is a design style that combines bold typography, bright colors, and intentionally 'ugly' or anti-aesthetic elements to create visually striking interfaces.

How to customize themes?

You can easily customize themes by modifying the CSS custom properties in the theme definitions. Each theme has its own color palette and styling rules.
